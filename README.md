<h1 align="center">Nysh Shell</h1>

<p align="center">

Nysh is a simple Unix-like shell written in C.  
It allows basic command execution and custom built-in commands.
</p># Nysh Shell


---

#  Features, Commands

Nysh currently supports the following builtin commands:

```bash
spec
whereami
wipe
hopon
die
say
self
selfroot
sos
nyamatrix
respawn
ghost
```

You can view all available commands inside the shell using:

```bash
sos
```

---

# Warning

This shell is in early development (v0.01.0).  
It is not stable and should not be used as a primary shell.
